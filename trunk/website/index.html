What is Celerity?

Celerity is a Ruby library for easy and fast automation of web application testing. Celerity wraps HtmlUnit via JRuby and is supposed to provide the same API and functionality as Watir.

Features

  • Fast: No time-consuming GUI rendering or unessential downloads
  • Scalable: Enables distributed multithreading
  • Unintrusive: No browser window interrupting your workflow (runs in background)
  • Easy to use: Simple API
  • Portable: Cross-platform

Background

FinnTech is a software company which develops software for the top providers of online classifieds in Europe. Consequently automated web testing is an essential part of our quality assurance effort. As of spring 2008 our Watir test suite, consisting of 340 test cases (which didn't really cover a substantial part of our web application), completed in 11 hours. Obviously, not ideal for an agile development environment.

We needed a faster alternative. At the same time, we enjoyed working with Watir's API. By wrapping the Java library HtmlUnit to Watir's API, we got rid of time-consuming GUI rendering and unessential downloads, while avoiding rewriting the existing test suite.

Installing Celerity

                
sudo gem install celerity

Code example

                
require "celerity" 

@browser = Celerity.new
@browser.goto('http://www.google.com')
@browser.text_field(:name, 'q').value = "Celerity" 
@browser.button(:name, 'btnG').click

assert(@browser.text.include?('celerity.rubyforge.org'))
